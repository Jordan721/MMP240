About MMP 240

In this class, I have learned how to design websites using CSS, Javascript, and Bootstrap. To make a responsive and well-made site. Effective website desing and site architecture will be explored through class assignments and critique of existing websites. Hands-on-experience designing web pages while using web authoring software and coding HTML manually will be emphasized. Graphic, audio, and animation applications, which allow for image and sound development, will be introduced.
